# 毕业论文参考文献扩充版

## 使用说明

本清单用于后续毕业论文排版与引用整理，当前已经尽量按学校给出的 ISO690 风格整理。正式提交前建议再做两项校对：

1. 按正文首次出现顺序重新编号。
2. 对图书类文献的出版地、版次和个别电子文献更新时间做最终核验。
